# Business Management Web App

A web-based ERP-style application for managing business operations.
Built with Laravel and Bootstrap 5.

## Features
- User authentication and role-based access
- Order management
- Warehouse and inventory tracking
- Nomenclatures and document management
- Configurable settings per company

## Tech Stack
- **Backend:** Laravel (PHP)
- **Frontend:** JavaScript, Bootstrap 5, SCSS
- **Libraries:** Axios, Tom Select, Bootstrap Icons
- **Build tool:** Laravel Mix
